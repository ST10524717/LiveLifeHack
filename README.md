# LiveLifeHack
//About the app:
LiveLifeHack App Overview LiveLifeHack is a simple Android quiz app that helps users test whether common life hacks are true or just myths. The app uses flashcards to make learning more interactive and engaging. How the App Works When the app starts, the user sees a welcome screen. After pressing the Start Quiz button, a statement is shown on the screen. The user must choose whether it is a Hack or a Myth. The app then shows if the answer is correct or incorrect. The user can move to the next question using the Next Question button. At the end, the app displays the total score and feedback. Features - Start screen with instructions - Flashcard-style questions - Hack or Myth answer options - Feedback after each answer - Score tracking - Final results screen - Option to restart the quiz Tools Used - Kotlin - Android Studio - XML Notes This app was created as part of a programming assignment to demonstrate basic Android development, including user input, arrays, and conditional logic.
